# Lsbonafe Tap

## How do I install these formulae?

`brew install lsbonafe/tap/<formula>`

Or `brew tap lsbonafe/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "lsbonafe/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
